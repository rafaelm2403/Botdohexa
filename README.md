#BotDoHexa

- Projeto

Esse projeto consiste em um chatbot no qual responde quaisquer dúvidas acerca do tema Copa do Mundo FIFA™. 
Foi feito a utilização de CSS, HTML, Python, Django e Flask para a construção deste.

- Utilização

No momento ele apenas é rodado em sua máquina local. Mas, se for de interesse, pode-se treinar ele para mais perguntas, não apenas com o tema de Copa do Mundo; Isso
vai da criatividade do dev em questão no qual precisar utilizar o projeto. 
Facilmente implementável em sites, blogs, aplicativos de mensagem (como o Telegram) e demais outras funcionalidades.

- Apenas o Escopo do Projeto

Esse é un projeto incompleto, ainda. Ele será implementado e incorporado atualizações, implementando um sistema mais completo e robusto.
